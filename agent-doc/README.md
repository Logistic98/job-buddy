## 文档维护规则

接口、配置、端口、服务职责、数据存储或用户流程发生变化时，应同步更新对应主题文档。主题文档必须说明正式设计、模块与接口、异常和安全边界及验证方式；不得包含迭代过程、路线图、真实密钥、默认密码、个人数据或未经代码验证的能力。图表优先使用兼容低版本 Mermaid 的 `graph LR` 和基础 `sequenceDiagram`，复杂流程拆分表达，并在提交前校验语法和仓库内链接。
